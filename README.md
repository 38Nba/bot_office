Телеграмм бот для бронирования мест в офисе.

Основные команды и их функционал
1. /start
 Описание: Приветственное сообщение.
 Функционал: 
  - Рассказывает пользователю, как пользоваться ботом:
  - Бронирование (/book)
  - Отмена брони (/cancel)
  - Просмотр своих бронирований (/mybookings)
  - Информация о доступных командах

2. /book <дата(ДД.ММ.ГГ)> <место>
 Описание: Забронировать место на указанную дату.
 Функционал:
  - Проверяет правильность формата даты.
  - Конвертирует дату в формат ГГГГ-ММ-ДД.
  - Проверяет, что дата не прошла.
  - Проверяет, что выбранное место свободно на эту дату.
  - Проверяет, что у пользователя еще нет брони на эту дату.
  - Создает новую бронь в базе данных.
  - Подтверждает успешное бронирование.

3. /cancel <дата(ДД.ММ.ГГ)> <место>
Описание: Отменить существующую бронь.
Функционал:
  - Проверяет правильность формата даты.
  - Конвертирует дату в формат ГГГГ-ММ-ДД.
  - Проверяет, что дата не прошла.
  - Ищет бронь пользователя на указанную дату и место.
  - Удаляет бронь из базы данных.
  - Подтверждает отмену.

4. /mybookings
Описание: Просмотреть все текущие брони пользователя.
Функционал:
  - Получает все брони пользователя из базы данных.
  - Форматирует список дат и мест.
  - Выводит список бронирований или сообщение о их отсутствии.

5. /admin_book <телеграмм_ник> <имя> <дата(ДД.ММ.ГГ)> <место>
Описание: Административная команда для добавления брони кому угодно (например, для ручного внесения).
Функционал:
  - Проверяет права администратора по ID пользователя, вызвавшего команду.
  - Создает бронь для указанного пользователя (по никнейму), с заданной датой и местом.
  - Автоматически определяет номер недели для этой даты.
  - Подтверждает добавление брони.
  - Итог

Этот бот позволяет пользователям:
  - Бронировать места на определённые даты
  - Отменять свои брони
  - Просматривать свои текущие брони
  - А администраторам — вручную добавлять брони для других пользователей через команду /admin_book.
